About This Site About the Copyright Compendium Viewer This site provides a navigable, searchable web display of the U.S. Copyright Office's Compendium of U.S. Copyright Office Practices, Third Edition. The Compendium is the administrative manual of the Register of Copyrights and provides expert guidance to copyright applicants, practitioners, scholars, the courts, and the general public regarding copyright law and institutional practices. 1 Experimental Transformation This is an experimental transformation of the official Copyright Compendium from PDF format to HTML for web display. The Compendium is officially published by the U.S. Copyright Office as PDF files, which can be challenging to navigate online. This project aims to make the Compendium more accessible by converting it into a web-friendly format with enhanced navigation and search capabilities. The transformation from PDF to XHTML was accomplished using advanced Large Language Models (LLMs), primarily Google Gemini 2.5 Pro Experimental. Each chapter of the Compendium was processed to extract and convert the text while preserving: Document structure and hierarchy Section numbering and cross-references Internal hyperlinks to other chapters and sections External hyperlinks to legal citations and copyright.gov resources Formatting such as italics, lists, and emphasis While every effort has been made to ensure accuracy, this is an unofficial, experimental version. For authoritative information, please consult the official PDF version on copyright.gov.
2 Quality Assurance and Text-Checking To ensure the accuracy of the transformation, an automated quality assurance engine has been developed to validate the converted content. This QA system compares the HTML output against the original PDF text to detect any discrepancies or conversion errors. The text-checker employs two complementary approaches: Algorithmic validation: Character-level diff comparisons between the source PDF text and the resulting HTML content LLM-based review: AI-powered analysis using the Gemini API to identify semantic differences and quality issues Each identified issue is classified by severity (HIGH, MEDIUM, or LOW) to help prioritize corrections. A GitHub Actions workflow automatically runs these quality checks whenever HTML files are updated, ensuring ongoing accuracy as the site evolves. For detailed information about the QA methodology, see the QA documentation and test suite in the project repository.
3 Purpose of This Site The Copyright Compendium is an essential resource for anyone working with U.S. copyright law, but the official PDF format presents several usability challenges: Difficult to navigate between chapters and sections Limited search functionality Not optimized for mobile devices Challenging to link to specific sections from external sources This web viewer addresses these challenges by providing: Easy navigation: A hierarchical sidebar shows the structure of each chapter, making it simple to jump to any section Powerful search: Algolia-powered search enables finding specific terms and sections across the entire Compendium Responsive design: The interface adapts to desktop, tablet, and mobile devices using the U.S. Web Design System (USWDS) Direct linking: Each section has a stable URL that can be bookmarked or shared Preserved hyperlinks: All cross-references and citations from the original PDF remain functional Experimental translation: Browser-based translation support for 12 languages (where supported by browser) By making the Compendium more accessible and user-friendly, this project aims to support copyright applicants, legal practitioners, researchers, and anyone seeking to understand U.S. Copyright Office practices.
